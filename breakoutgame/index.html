Brick Breakout

Welcome to brick break out. Please see the control list below:

  • Press space to start the game
  • Press esc to pause/unpause the game
  • Press the left arrow key to move the paddle left
  • Press the right arrow key to move the paddle right

This game is a work in progress so please check back for updates! :)

Brick types:

  • Orange bricks are standard bricks and have no effect on the game play
  • Blue bricks will double the player paddle width (once per life)
  • Coming soon: green bricks will double the speed of the ball (once per life)
