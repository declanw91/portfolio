Online Seller Blog System

This was originally an assessment as part of an E-Commerce module. The task was to create a blog system that allowed an admin to attach files to a post so they could be sold to users. Only users with an account are able to see and buy the files. The project uses PHP, MySQL, HTML and CSS. It uses a PDO object to access the database of users and posts. The payment system is an integration of the Google Checkout libraries.

Viewable here: Blog Store
